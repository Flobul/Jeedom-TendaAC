### Fonctions disponibles

Infos :
* Etat du routeur
* Etat du Wifi
* Nom du routeur
* Nom des SSiD WiFi
* Version de firmware

Actions :
* Reboot
* Backup

Scénarii possibles :
* Sauvegarder la configuration à intervales réguliers
